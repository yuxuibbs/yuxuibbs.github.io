Academic Games Projects

Online Academic Games Practice Tool

HTML, CSS, JavaScript, jQuery

Study tool for Academic Games players to help players study old Academic Games tournament questions for the games Presidents and Propaganda and practice various Equations tricks

Academic Games Player

Python, Python Flask

Solution finding application for the games LinguiSHTIK and WFF N’ Proof for coaches and students to use as an educational tool

For the game LinguiSHTIK: Enter a list of demands for the game LinguiSHTIK and the program will return a sentence that fulfills those demands.

For the game WFF 'N Proof: Enter the premises and the goal and the program will return a proof.
